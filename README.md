# Sticky Notes

An easy on eye, colorful and alluring note taking app for all


## Functionality

* Create Todo Notes
* Update Notes
* Delete Notes
* View personal Notes
* Search Notes
* Mark notes as completed
* Share notes with friends and many more

## Tools & Libraries

* Django web framework
* Bootstrap
* Jquery
